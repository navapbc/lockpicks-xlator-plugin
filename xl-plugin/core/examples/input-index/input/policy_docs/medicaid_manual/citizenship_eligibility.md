# MS 521. CITIZENSHIP AND ALIEN STATUS (Medicaid)

## A. Citizenship requirement

An applicant must be a U.S. citizen or a qualified non-citizen to be eligible for Medicaid.

## B. Qualified non-citizen categories

The following non-citizen statuses qualify as "qualified non-citizen" for Medicaid:

- Legal Permanent Resident with five or more years of residency
- Refugee
- Asylee
- Cuban / Haitian entrant
- Victim of trafficking
- Iraqi / Afghan special immigrant

## C. Verification

Citizenship or qualified-non-citizen status is verified through SAVE or, when SAVE returns indeterminate, through caseworker-collected documentation. When verification cannot be completed within the application window, the case is set to `further_verification_required` rather than denied outright.
