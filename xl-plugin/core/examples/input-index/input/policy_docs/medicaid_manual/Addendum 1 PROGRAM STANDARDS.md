|---------------:|-------:|------:|-----------------:|-------:|-------:|-------:|
| 1              | $1,255 | $1,255 | $2,510           | $1,255 | $1,506 | $2,510 |
| 2              | $1,850 | $1,850 | $3,700           | $1,704 | $2,045 | $3,408 |

| 8              | $4,730 | $4,730 | $9,460           | $4,392 | $5,273 | $8,784 |

For households of more than 8, add $480 per additional member (Family / Child / QMB / SLMB), $960 per additional member (Working Disabled / QDWI).

## State Supplement

When the applicant qualifies for the Alaska State Supplement (residency-dependent), add the per-household-size supplement amount from the State Supplement column in the row matching the household_size.
