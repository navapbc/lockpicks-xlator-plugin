# Traffic Violation Code

**Effective Date:** January 1, 2026
**Issuing Authority:** Department of Motor Vehicles

## Section 100 — Purpose and Scope

This Code establishes the classification of traffic violations, the demerit points assigned to each violation, and the schedule of monetary fines. It applies to all holders of a Class C (standard) or Class M (motorcycle) driver's license operating a motor vehicle on public roadways.

## Section 200 — Definitions

**Violation.** Any act or omission by a driver that contravenes a provision of this Code.

**Severity Class.** The category assigned to a violation based on its risk to public safety, as set forth in Section 300. Severity Class is a property of the violation itself and does not vary by driver history.

**Demerit Points.** The numeric value added to a driver's record upon adjudication of a violation, as set forth in Section 400.

**Base Fine.** The monetary penalty for a violation before any surcharges or multipliers, as set forth in Section 500.

## Section 300 — Severity Classification

Every violation under this Code shall be classified into exactly one of the following Severity Classes:

- **Class A — Minor.** Violations creating low immediate risk to other road users.
- **Class B — Moderate.** Violations creating elevated risk, typically involving disregard of traffic controls or unsafe operation.
- **Class C — Serious.** Violations creating substantial risk of injury, including excessive speed and reckless conduct.
- **Class D — Major.** Violations creating grave risk of injury or death, including impaired operation and willful endangerment.

The Severity Class assigned to each enumerated violation is fixed by this Code and shall be used consistently wherever Severity Class is referenced, including in the License Suspension Policy.

## Section 400 — Demerit Points

Upon adjudication of a violation, the demerit points assigned shall be determined by the Severity Class of the violation:

| Severity Class | Demerit Points |
|----------------|----------------|
| A              | 1              |
| B              | 3              |
| C              | 5              |
| D              | 8              |

Points are recorded against the driver's license on the date of adjudication.

## Section 500 — Fine Schedule

The Base Fine for a violation shall be determined by the Severity Class of the violation:

| Severity Class | Base Fine (USD) |
|----------------|-----------------|
| A              | $50             |
| B              | $150            |
| C              | $350            |
| D              | $750            |

### Section 510 — School Zone Multiplier

If a violation occurs within a posted school zone during posted school hours, the Base Fine shall be multiplied by 2. Demerit points are not affected by this multiplier.

### Section 520 — Repeat Violation Surcharge

If the driver has been adjudicated of any prior violation of the same Code (e.g., a second TV-07) within the preceding twelve (12) months, an additional surcharge of $100 shall be added to the Base Fine. Demerit points are not affected by this surcharge.

## Section 600 — Payment and Adjudication

A driver may pay the assessed fine within thirty (30) days of issuance, which constitutes adjudication. Contested citations are adjudicated by the Traffic Court. Demerit points are recorded only upon adjudication, not upon issuance.
